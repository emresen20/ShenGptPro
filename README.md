# ShenGptPro

ShenGptPro is a React Native ai app using Chatgpt:


- Expo
- Vectoricons
-  Navigation
- Fetch-axios api
- react-native-dotenv
- GiftedChat
- MaterialIcons
- useState
- ChatGpt Developer
